== Description ==

**BuildWithGuru** is a custom header and footer plugin designed for Elementor. It includes a sticky header feature, allowing the header to stay visible as users scroll. You can enable the sticky header across the entire site or only on selected pages, making navigation easy and convenient. BuildWithGuru gives you a professional look with flexibility for unique page customization.

== Features ==

* Custom header and footer builder
* Sticky header functionality
* Option to apply sticky header to specific pages or site-wide
* Fully compatible with Elementor

== Installation ==

1. Upload the `buildwithguru` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Elementor and start creating your custom header and footer with the sticky header options.

== Frequently Asked Questions ==

= Can I use this plugin without Elementor? =
No, BuildWithGuru is built to work with Elementor and requires Elementor to be installed.

= Can I control which pages show the sticky header? =
Yes, you can choose to show the sticky header on specific pages or across the entire website.
