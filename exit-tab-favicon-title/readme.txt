== Short Description ==
Easily change the favicon and title of your website when users exit the tab or the page becomes hidden.

== Description ==
Change the favicon and title of your website when users exit the tab, drawing their attention back to your site when they return.

== Installation ==
1. Upload the `exit-tab-favicon-title` plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Exit Tab Favicon and Title to configure the settings.
